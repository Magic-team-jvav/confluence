# 更新日志（1.2.0）：

## 注意事项：

- 血肉之山（注意不是血肉之墙）会清空大范围场地，请一定要远离玩家活动区召唤（越远越好）
- 额外物品栏已进行重构，如果担心物品丢失，请先备份存档
- 除红绿蓝染料，回声漆的临时配方外，其他染料/油漆的临时配方已经删除，转为染缸处合成
- 困难模式（默认转变策略对于设备会有明显卡顿且时间较长，可以从配置中改为立刻转变，届时tps会完全停止但时间较短）
- 未完成物品现在默认不会显示在创造模式标签页里，需要手动打开配置

- 部分生物模型z闪
- 花岗岩与大理石洞穴生成破洞
- 蠕虫y轴旋转卡闪、蠕虫头卡顿
- 队伍按钮时常消失
- 再生法杖最大堆叠为1，能随意刷萝卜、土豆之类的作物，即使它们没有成熟，无法收获未成熟草药
- 事件同步到错误的玩家
- 手持长矛交互导致攻击
- 射出的子弹的崩溃问题
- 由于某些情况导致配饰重复佩戴而产生的崩溃
- 史莱姆攻击不到玩家
- 人形怪不渲染靴子
- 杀死仆从会掉落掉钱币的问题
- 地牢祭坛的墓碑中的字没有本地化
- 世界吞噬者会淹死、飞鱼会摔伤的问题
- 使用魔法枪械消耗魔力，导致魔力花失效的问题
- 修复套装效果属性丢失的Bug
- 枪械连续吃两次伤害加成的问题
- 钩爪能被水流冲走的问题
- 巨石可以杀死触发了不死图腾的玩家
- 锤子无视破坏进度直接破坏
- 钱币、魔力星拾取范围能力失效
- 副手持有盾牌时潜行右击祭坛无效的问题
- 香蒲生成错位的问题
- 邪恶冰块不传播、猩红不传播原木和树叶
- Boss可作为旗帜的问题
- 悠悠球不能攻击蠕虫
- 攻击蠕虫身体来杀死蠕虫不会掉落战利品
- 敌怪旗花纹渲染错误，且不能掉落
- 魔力消耗等于0时也会设置魔力再生延迟的问题
- 玩家退出游戏后，已完成的阶段复原的方块属性，进入另一个没完成阶段的档时，属性没重置的问题
- 椅子乘坐位置错误的问题
- 虫网捕捉坐在载具上的生物时崩溃
- 虫子，鸟能被摔死的问题
- 地物中的毒镖有时候会越界生成并造成卡顿的问题
- 宝匣不再能被烧毁
- 鱼饵错误的消耗概率
- 星星炮没有无限穿透

## 优化：

- 部分词缀属性显示的位置
- 召唤百分比增伤可应用于召唤物
- 仅第一次解锁旗帜才告诉玩家“该旗帜已在旗帜盒中解锁”
- 无法破坏的物品现在可以在铁砧内合并附魔
- 允许魔法武器直接在附魔台里附魔
- 模型贴图、合成配方、模组性能、物品价值、交易列表、成就触发、游戏细节
- 检测药水，检测饰品列表，生物追踪配置增加更多内容
- 洞穴探险(spelunker)效果不会显示被隐藏的方块
- 被隐藏的方块现在也会对调试界面(debug screen)隐藏
- 金属探测器(metal detector)不再检测被隐藏的方块
- JEI显示环境合成条件
- 瓶子(confluence:bottle)与水瓶(confluence:bottled_water)可以与炼药锅(cauldron)交互
- 抓人草根部渲染偏移
- 腐化洞穴现在沿着地表生成
- 药水物品与药水效果现在有功能提示
- NPC的对话按钮
- 玩家重生或换维度后阶段数据不同步
- 史莱姆王平滑缩放
- 世界吞噬者游走AI，且部位分帧生成
- 优化事件触发器的逻辑
- 物品稀有度颜色现在兼容传说提示框(Legendary Tooltips)与物品边框(Item Borders)
- 部分工具武器将在物品提示内显示其高清大图（同时安装传说提示框时，不会显示大图）
- 实体追踪(EntityTracker)渲染体系完全重构，实验性加入了第二种指针模式(目标锁定型，默认配置页面关闭)。
  - 所有配置都可以通过暂停界面的 模组>EntityTracker>配置... 打开的配置屏幕进行修改。
  - 目标锁定型使用较中心环绕型更加复杂的配置语法。
  - 允许玩家以材质包的形式添加/修改贴图，识别到的贴图可以通过在配置直接输入对应文件名(无后缀)应用。
  - 增加指针缓动插值算法，添加指针大小配置、仅追踪最近n个实体设置和刷新频率配置，并使每种指针开关独立。
  - 新增的特殊指针目前为目标锁定型指针独有，在0.1.1(ETHUD当前版本)是彩蛋。
  - 在开启中心环绕型指针后，可以尝试在对应追踪列表添加`minecraft:villager|3body:normal_white|rainbow:2`，
    该彩蛋将会使指针变成彩虹指针并环绕附近村民指示。
- 稀薄蜂蜜块不再遮挡玩家视线
- 额外栏(extra inventory)的信息配饰开关按钮
- NPC离开家（没有家就选择其出生点）64米远会自动传送回来（有玩家在其32米范围内时不传送）
- 房屋检查、设置或移除现在可通过额外栏中的按钮
- 事件系统渲染效率
- 重填洞穴探险效果显示颜色
- 炼药台泰拉药水配方的配置移到启动时(startup)，且默认为开，并为其添加JEI支持
- 祭坛现在允许一次性拿完单个物品堆
- 粘性炸弹浮空的问题
- 部分剑改为特殊横扫范围（以玩家为中心，半径为实体交互距离的扇形），且允许对着空气挥动触发横扫
- 部分剑允许自动挥舞，且自动挥舞的剑都为特殊横扫
- 狱火药水，更多射弹带有粒子显示，优化法杖粒子反馈
- 补充更多物品价值（包括MC原版部分材料）
- 永夜刃与草剑现在可以攻击末影人

## 调整：

- 限制蠕虫类、飞龙(wyvern)的游走高度和索敌高度
- 特殊子弹提高伤害
- 增加史莱姆王宝藏袋中的忍者套装抽取次数
- 下调气球河豚鱼(balloon pufferfish)、蛙腿(frog_leg)与金鲤鱼(golden carp)的运气影响
- 护士不再主动治疗玩家，且可以治疗减益(debuff)
- 调整怪物防御
- 史莱姆王可以跳四次、专家模式（MC普通模式）血量较低时可以跳两轮
- 蜂后只能平飞、小改蜂后AI、限制其仆从数量
- 略微下调生命水晶的生成数量
- 钓竿使用鱼饵的方式从收杆时使用，改为放线时；并把鱼饵显示在鱼竿物品右下角
- 永夜刃(nights_edge)的攻击范围扩大至原来的三倍
- 蜜蜂射弹（来自蜂巢配饰、蜜蜂手雷）改为静态无敌帧
- 血月事件中原版怪不在无视亮度生成
- 地下矿井陷阱率降低
- 完善部分事件的细节
- 下调恶魔之镰(demon_scythe)的伤害
- 加快幽匿游灵(sculk_wisp)的攻速
- 反转重力键位改为↑，允许反转重力跳跃
- 提升部分剑的伤害，降低部分剑的攻速
- 现在羊毛转化为线的合成下调至2
- 模组内炸弹使用泰拉样式爆炸（可通过配置开关）
- 现阶段除木箭外的所有箭矢都会落地消失
