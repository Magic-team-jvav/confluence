# 生物生成数量系数

该属性会影响玩家附近的自然生成数量，且随距离衰减。详细影响方式如下：

> 首先服务器会获取各种**生物分类**(即`MobCategory`)定义的生物自然消失距离，并取其中的最大值。然后通过该公式计算得到最大衰减距离：

`最大衰减距离 = 最大自然消失距离 / 16 + 1`

> 然后对玩家周围的区块以曼哈顿距离进行衰减，衰减系数为：

`衰减系数 = 1 - 目标区块距离玩家所在区块的曼哈顿距离 / 最大衰减距离，大于1则取1，小于0则取0。`

> 通过该衰减系数，计算得到所有玩家属性值叠加得到的区块属性值：

`区块属性值 = (玩家A的属性值 * 玩家A的衰减系数 - 1) + (玩家B的属性值 * 玩家B的衰减系数 - 1) + ...`

> 最后，根据衰减系数计算得到最终自然生成数量：

`最终自然生成数量 = 原始自然生成数量 * 区块属性值`

::: warning
过大的属性值会导致游戏严重卡顿！
:::
