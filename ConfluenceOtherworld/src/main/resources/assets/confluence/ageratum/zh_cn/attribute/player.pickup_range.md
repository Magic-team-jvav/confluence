# 拾取范围

定义了玩家可以隔空吸取物品的范围，单位为米。

## 数据

|   默认值    |                                      0.0                                      |
|   最小值    |                                      0.0                                      |
|   最大值    |                                     64.0                                      |
| 推荐的修饰符操作 |                                   ADD_VALUE                                   |
