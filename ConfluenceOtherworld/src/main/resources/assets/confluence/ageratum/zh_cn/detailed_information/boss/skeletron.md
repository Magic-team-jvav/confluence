# 骷髅王：

- 攻击18.2 血量2288 防御10（专家模式）
  攻击额外无视玩家4点防御


- 骷髅王是一个困难模式之前的 Boss。玩家在进入[地牢](dungeon.md)
  之前必须击败骷髅王，击败骷髅王可以让所有玩家在该世界获得任意进出地牢区域的权限。击败它也可以使服装商
  NPC 在有空余房屋的时候入住。
- 骷髅王在白天会进入狂暴的状态，其表现会变得像地牢守卫
- 生成时，其会携带俩只骷髅王手
- 当手被全部摧毁时，骷髅王防御降低至0

## 骷髅王手：

- 攻击10 血量405 防御4（专家模式）
  攻击额外无视玩家4点防御

## 生成条件：

- 与地牢老人在夜晚对话生成
- 装备服装商巫毒玩偶后击杀服装商召唤

## 脱战：

- 骷髅王会在丢失目标一段时间后消失（可在泰拉生物模块配置是否脱战）

### 技能一：

- 朝向玩家头顶 5 格的方向移动
- 这个移动具有加速度和最大速度

| 难度 | 加速度  | 最大速度 |
|----|------|------|
| 经典 | 0.07 | 0.7  | 
| 专家 | 0.1  | 1.0  | 
| 大师 | 0.1  | 1.0  | 
| 传奇 | 0.16 | 2.0  |

- 30% 概率进入疯狂模式：额外向玩家方向加速
- 释放技能时有 50% 概率切换仇恨目标

### 技能二：

- 朝向玩家方向旋转头颅攻击
- 经典：固定速度：0.2
- 专家 / 大师：速度公式：0.01 × 距离 + 0.16，限幅 0.22 ~ 0.48
- 传奇：再在大师的基础上 × 1.3
- 剩1只手：× 1.05
- 无手× 1.1
- 在白天狂暴时，固定极速：1.0

### 技能三：

- 当专家以上时、非旋转时，血量<75% 或 手<2 只时会玩家发射，缓慢追踪的骷髅头弹幕攻击
- 伤害：6 点
- 基础冷却：20 tick
- 无手时冷却：10 tick
- 传奇： 8 tick

### 技能四：

- 骷髅王会使用它的两只手，当手到达预定的范围时会对玩家进行突刺拍击
