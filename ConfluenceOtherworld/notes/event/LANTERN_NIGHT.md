# 灯笼夜

## 输出信息

## 触发条件（玩家）：

- [x] 第一次击败任意 Boss。（这个触发条件的灯笼夜是“非随机的灯笼夜”）
  - [ ] 但除了哀木、圣诞坦克、常绿尖叫怪
- [x] 第一次完成任意入侵。（这个触发条件的灯笼夜是“非随机的灯笼夜”）
  - [ ] 不过，第一次击败南瓜王或冰雪女王会开始一场灯笼夜。
- [x] 当击败月亮领主后，灯笼夜会随机发生，几率为 1/14 (7.14%)。（这个触发条件的灯笼夜是“随机的灯笼夜”）

## 事件冷却

- [x] 在随机的灯笼夜发生后，下次随机的灯笼夜要直到 5-10 天后才能发生。

## 触发条件（时间）：

- [x] 下午7:30

## 触发条件（概率）：

- [x] 1/14（随机灯笼夜）

### 触发概率表格：

## 事件表现：

- [ ] 在事件期间，背景里会有许多灯笼不断飘上天。
- [x] 玩家的运气会增加 0.3
- [x] 派对女孩会售卖放飞灯笼。

## 结束条件：

- [x] 持续时间结束
- [x] 在灯笼夜时召唤boss或事件

### 持续时间表

- [x] 下午7:30 ~ 上午 4:30

## 额外注意事项

- [x] 同一时间只会安排一次非随机的灯笼夜。因此，如果在 24 小时内有两场灯笼夜被触发（从一天的夜晚降临到下一天的夜晚降临），只会实际安排一次灯笼夜。
- [x] 激活的灯笼夜会阻止克苏鲁之眼，且持续到黎明的灯笼夜会阻止哥布林入侵。
  - [ ] 阻止海盗入侵、日食、任何机械 Boss 自然生成

- 灯笼夜可以被其它的事件预先覆盖。
  - [x] 若当晚本应是灯笼夜，但在夜晚降临时有 Boss 存在，或是血月正在发生，那么当晚则不会出现灯笼夜
    - [ ] 或是南瓜月、霜月、或月亮末日正在发生。
  - [x] 当上述情况发生时，下一个夜晚将会是符合要求的灯笼夜。因此，当灯笼夜被预先覆盖时，它会被推迟，而非取消。
  - [x] 类似地，月亮领主后的随机灯笼夜也可以预先覆盖非随机的灯笼夜：如果一个夜晚符合随机灯笼夜的要求并且已经预先安排了一场非随机的灯笼夜，若
    1/14 (7.14%) 几率的随机成功，就会发生一场随机的灯笼夜，非随机的灯笼夜则会继续留在安排中，因而会在下一夜晚发生。
