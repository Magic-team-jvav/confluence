# 哨兵容量

定义了玩家可以召唤的哨兵数量。

::: info
哨兵数量为其属性值向下取整得到的数
:::

## 数据

|          |                                                                                  |
|:--------:|:--------------------------------------------------------------------------------:|
|   内部键    | <confluence_hover>confluence_magic_lib:player.sentry_capacity</confluence_hover> |
|   翻译键    |    <confluence_hover>attribute.name.player.sentry_capacity</confluence_hover>    |
|   默认值    |                                       1.0                                        |
|   最小值    |                                       0.0                                        |
|   最大值    |                                      128.0                                       |
| 推荐的修饰符操作 |                                    ADD_VALUE                                     |
|    类型    |     <hover type="SHOW_TEXT" data="增加正面属性值的修饰符为蓝色，减少正面属性值的修饰符为红色。">正面</hover>     |
