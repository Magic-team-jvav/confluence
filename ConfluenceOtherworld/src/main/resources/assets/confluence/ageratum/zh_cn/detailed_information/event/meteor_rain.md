# 陨石雨

## 信息提示

- 无信息提示

## 触发条件（概率）：

- 它会在激活了 The Constant 和 For the Worthy 种子，且未激活 Remix 种子的世界中的雷雨期间发生。

## 事件表现：

- 期间会有陨石射弹从天空落下。流星对玩家造成 37 / 74 / 109 伤害（对非玩家实体造成 18 伤害）并造成极强击退。这些陨石射弹在消失时不会掉落任何物品。
- 每刻有20%的概率（醉酒世界为40%）落下随机方向的陨石射弹，风速越高，随机偏移的范围更大
## 结束条件：

- 它在钱币总价值达到一个隐藏值时停止，累积掉落钱币1铂50金-3铂的范围
- 当雨停止，或其转进为雷暴鱼时该事件也会提前停止。
- 当世界中已存在过多物品实体时该事件也会被强制停止。
