# 套装效果


挖矿盔甲：
- 照明头盔：提供照明
- 护腿：+10%挖矿速度
- 胸甲：+10%挖矿速度

粉色，蓝色防雪套：集齐四件可以免疫冷冻之类的debuff，同暖手宝


渔夫盔甲：集齐三件套降低敌怪生成速度（减半） ，每一件渔夫盔甲都能加5%渔力


仙人掌盔甲：集齐四件，仙人掌套简单，普通，困难反伤3，8，11点伤害


忍者盔甲：四件套，每一件都能给玩家增加2暴击率，集齐获得20%移速提升



化石盔甲：
- 头盔和腿，提升4%远程暴击率，
- 胸甲和鞋子，提升2.5%远程伤害
四件套集齐时，提供20%几率不消耗弹药的套装效果

蜜蜂盔甲：
- 头盔：召唤伤害提高4%，仆从数量上限增加1
- 胸甲：召唤伤害提高4%，
- 腿：仆从数量上限增加1
- 鞋子 召唤伤害提高5%
  四件套集齐时，召唤伤害提高10%


孢根盔甲：
- 头盔：召唤伤害提高2%，仆从数量上限增加1
- 胸甲：召唤伤害提高3%，
- 腿：召唤伤害提高3%，
- 鞋子 召唤伤害提高2%
  四件套集齐时，仆从数量上限增加1

角斗士盔甲：
  四件套集齐时，赋予玩家免疫击退


陨石盔甲： 每一件都能独立增加7%魔法伤害 ，集齐四件套：获得让太空枪，光剑魔力消耗降低为0的套装效果（暂时没
有太空枪）


暗影套：每一件都能给玩家带来3.5%的暴击率，集齐四件套：+75%的加速度，1.15乘算的速度加成


血腥套：每一件都能提升玩家2%的伤害，集齐四件套时，提高再生速度（参考生命手环的速率）玩家不走动时，再生更快一点点


寒晶套：
- 头盔：最大魔力值增加20，魔法暴击率提升4%
- 胸甲：最大魔力值增加20，魔法伤害提升4%
- 护腿：魔法伤害提升4%
- 鞋子：魔法暴击率提升4%
集齐全套：魔法攻击会有附带霜冻debuff



欧泊（渊鳞）：每一件都能增加1%的近战伤害
- 头盔：延长水下呼吸时间5%
- 胸甲：成功使用护盾拦截下一次攻击后三秒内伤害提升20%
- 如果三秒加伤过程中你发动攻击命中，加伤停止，或者你被击中，加伤停止
- 护腿：增加2%暴击率
- 鞋子：提高水下移动速度5%
- 集齐四件套：赋予你4点血量的伤害吸收，每隔5分钟再次赋予

狱石套：
- 头盔：近战暴击率提高7%
- 胸甲：近战伤害提高7%
- 护腿：近战速度提高3.5%
- 鞋子：近战速度提高3.5%
- 集齐四件套：近战伤害提高10%，免疫着火
