# 哥布林军队

## 输出信息

- [x] 预备时输出““一支哥布林军队正在逼近！””（紫色）
- [x] 开始时输出“哥布林军队来了！”（紫色）
- [x] 结束时输出“哥布林军队已被打败！”（紫色）

## 事件冷却

## 触发条件（玩家）：

- [x] 世界中至少有一名玩家生命值大于40（原泰拉数值 200 ，吃过5颗生命水晶）。
- [x] (这一条是汇流加的)世界中至少有一名玩家防御值大于16（原泰拉没有这一条，考虑到汇流哥布林军队比较具有威胁）
- [x] 没有其他入侵事件
- [ ] 没有撒旦军队激活
- [x] 至少摧毁了一个暗影珠或猩红之心。

- [x] 哥布林战旗在满足条件(世界中至少有一名玩家生命值大于40)时可以用手动触发
  - [x] 如果没有玩家满足此条件或已经有入侵事件激活，哥布林战旗将没有效果且不会被消耗。

## 触发条件（时间）：

- [x] 上午4:30

## 触发条件（概率）：

### 触发概率表格：

| 条件 | 哥布林军队未被击败 | 哥布林军队已被击败至少一次 |
|----|-----------|---------------|
| 肉前 | 1/3       | 1/30          |
| 肉后 | 1/3       | 1/60          |


## 事件表现：

- [x] 不再“自然”生成其它生物

## 结束条件：

- [x] 哥布林军队事件在击杀足够多来自此事件的敌怪后即完成。

### 计分细则
- [x] 所需击杀数和事件开始时世界中具有至少 40 最大生命值的玩家数量成正比，可以通过下方公式计算：
  - 击杀数 = 80+40x玩家数量(这些玩家具有至少40最大生命值)

- 例如：
  - 在单人模式中，所需击杀数为 80+40x1 =120
  - 在多人模式中有 8 个满足条件的玩家时，所需击杀数为80+40x8 =400

- [x] 一次入侵的最高击杀数为 10,280（255 个玩家）。
- 大多数敌怪算作 1 次击杀，但某些敌怪可能算作更多或更少。详细见下表：
  - [ ] 哥布林术士算10次击杀
  - [ ] 暗影焰幻鬼不算入计数

## 额外注意事项

- [x] 击败哥布林军队后可使哥布林工匠生成
